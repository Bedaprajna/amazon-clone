# Amazon Clone

A responsive Amazon homepage clone built using **HTML5** and **CSS3**.

## Features

- Amazon-style navigation bar
- Search bar
- Hero banner
- Product category cards
- Responsive layout
- Hover effects
- Multi-column footer
- Clean and modern UI

## Technologies Used

- HTML5
- CSS3
- Font Awesome
